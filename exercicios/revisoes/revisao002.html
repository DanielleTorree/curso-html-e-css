Tags de hierarquia de parágrafo.

<h1> primeiro nível

<h2> segundo nível.

<h3> terceiro nível.

<h4> quarto nível.

<h5> quinto nível.

<h6> sexto nível.

Tag de força

Deixando a palavra mais forte com a tag <strong>.

Tag de ênfase

Deixando a paravra mais enfática com a tag <em>.

Tag de deletar texto

Delentendo uma palavra com a tag <del>.

Tag de inserir texto

Inserindo uma palavra com a tag <ins>

Você pode inserir uma palavra com a tag <u>, porém está tag não é semântica, já a tag ins é semântica.

Tag de citação

Aqui tem uma citação em bloco com a tag <blockquote cite="URL">.

Você ainda pode fazer outros tipos de citações, porém esta é mais usual para citações no meio do parágrafo. A tag é <q>.

Tag de marcação de texto

Marcando um texto com tag <mark>.

Tag de sobrescrito

Você pode escrever equações entre outras coisas com a tag <sup>, assim: x6+3x5-9x4+4x3+7x2-10x+3=0.

Tag de subscrito

Você pode escrever fórmula aquosa entre outras coisas, assim: H2O.

Links

Vamos aprender a "linkar" páginas.

Abrir link na mesma página

Conheça mais sobre a linguagem HTML no site W3C. A tag usada foi <a href="URL">

Abrir link em outra página

Conheça também o site W3C School, pois há nele diversos tutoriais sobre a linguagem HTML. A tag usada foi <a href="URL" target="_blank">

Abrir links das páginas do site

Navegue também nesta outra página do nosso site, neste link: Revisão 001. Para isso, utilizamos a tag <a href="revisao001.html">

ATENÇÃO! Como o link é para navegação dentro das páginas do nosso site, não precisamos usar a URL absoluta (o caminho inteiro), somente precisamos utilizar a URL relativa.
